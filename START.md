# 个性化大模型助手用户中心系统

## 项目启动说明

### 项目结构
```
├── index.html          # 主页面

├── profile.html        # 个人中心主页
├── knowledge.html      # 知识记忆管理页面
├── preferences.html    # 检索偏好设置页面
├── userinfo.html       # 个人信息管理页面
├── tools.html          # 专属工具集页面
├── css/
│   └── style.css       # 样式文件
├── js/
│   ├── auth.js         # 认证功能
│   ├── knowledge.js    # 知识管理
│   ├── preferences.js  # 偏好设置
│   ├── userinfo.js     # 用户信息
│   ├── tools.js        # 工具管理
│   ├── profile.js      # 个人中心
│   └── main.js         # 主应用逻辑
├── assets/
│   └── icons/          # 图标资源
└── README.md           # 项目说明
```

### 启动方式

由于这是一个前端项目，您有以下几种方式启动：

#### 1. 使用本地服务器（推荐）
```bash
# 如果您有Python环境
python -m http.server 8000

# 或者如果您有Node.js环境
npx serve .

# 或者使用live-server
npm install -g live-server
live-server
```

#### 2. 直接在浏览器中打开
将 `index.html` 文件拖拽到浏览器中打开即可。

### 功能特性

1. **用户认证系统**：支持账号密码登录、邮箱验证登录、注册功能
2. **知识记忆管理**：记录与大模型的交互历史，支持分类、检索、编辑
3. **检索偏好设置**：管理常用网站，设置检索优先级
4. **个人信息管理**：管理个人资料与文件上传
5. **专属工具集**：集成个性化工具，支持前端隔离
6. **主题切换**：支持暗黑/明亮模式切换
7. **响应式设计**：适配各种设备屏幕

### 技术特点

- 使用原生JavaScript，无外部依赖
- 数据存储使用localStorage
- 模块化设计，代码结构清晰
- 响应式布局，支持多种设备
- 用户友好的交互体验

### 使用说明

1. 首次访问将跳转到登录页面
2. 可以使用邮箱和任意密码注册新账户
3. 登录后可以访问所有功能页面
4. 所有数据均保存在本地浏览器中
